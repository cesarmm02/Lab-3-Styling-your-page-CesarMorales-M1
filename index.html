Types of CSS

CSS is a way to customize your website. There are two kinds of CSS, internal and external.

Internal CSS is in the tag or in the head of the html document. If it's in the tag, it only changes that one instance.

External CSS has it's own .css file. Both External CSS and internal CSS in the head both change all instances of the tag.
